# My Capstone Project: MatchPal

Introducing MatchPal, the ultimate soccer app! Discover matches and bars to watch games with your friends. Explore an interactive map to locate nearby viewing spots. Leave reviews, share experiences, build a vibrant community. Never miss a moment of soccer excitement with MatchPal! Elevate your matchday experience!

## How to setup?

- Clone this repository
- Install all dependencies with `$ npm install`
- create a Cloudinary account and change the cloudname in the App.js
- insert your API_Key, API_Secret and Cloudname as an enviroment variable in the terminal
- Run app in dev mode with `$ npm run dev`
- Server: [http://localhost:3000/](http://localhost:3000/)
- Run tests via `$ npm run test`
